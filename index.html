📁 File Upload Proxy

Upload files via multipart/form-data and retrieve them.

📡 Endpoints

POST /api/upload

Accepts multipart file field file. Returns file info and a download URL.

GET /api/download?file=filename

Downloads the uploaded file (temporary, stored in /tmp).

🔑 Authentication

All endpoints require X-API-Key header.
